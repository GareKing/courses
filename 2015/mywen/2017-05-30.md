  今天学了课程《github使用》的第一部分的1.20到1.32的内容，主要包括以下几个方面：		
 * 介绍了克隆代码库的指令
```bash		
 git clone		
```		
  看到彩色结果的指令(方便阅读)
```bash		
 git config --global color.ui auto	
 git config（更改设置） --global（应用到所有git项目上） 
 ```		
  临时将文件返回到任何commit时的状态		
```bash		
 git checkout		
```		
 * 移动文件位置（更改文件名称）		
```bash		
    mv (要移动文件地址/名称) （要移动的新位置/新名称）
    不指定新位置的话，会自动保存在原位置
```		
 * 使用sublime查看bash配置文件
```bash		
 subl.bash_profile
```
