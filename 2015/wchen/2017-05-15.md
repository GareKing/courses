### 版本控制的作用  
* 查找文件之间的差异  
1.Windows  命令提示符中使用工具 FC（文件对比的意思）  
命令提示符终端-转到文件所在的目录（cd 文件夹）-输入文件比较指令-Enter  
Windows下的文件比较指令
```bash
FC a-old.file b.file
```
2.Mac或Linux 内置工具 Diff（表示差别）  
linux下的比较指令
```bash
diff -u a-old.file a.file
```
-u 表示标准区别格式 是输出内容更容易阅读  
+- 表示 增加或减少
