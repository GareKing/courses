## 2017-05-23今天看了视频4.1-4.5学习GitHub协作
* 使版本库保持同步
* 添加远程版本库                                                                        
1.查看所有当前远程代码库 
```bash
git remote
```
2.将GitHub上的代码库当做远程代码库
```bash
git remote add origin（可更改） +url
```
3.验证添加URL正确无误
 ```bash
git remote -v
```
4.向远程代码库发送更改，git push将所有可通过master分支到达的commit复制到origin
```bash
git push origin（与上面保持一致）master
```
