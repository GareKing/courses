## 2017-05-13
今天第一次正式学习《如何使用git和github》，已经看了从1.1到1.23的视频，其中比较重要的几个指令有：
* windows下的文件比较指令
```bash
FC a.file b.file
```
* linux下的比较指令
```bash
diff -u a.file b.file
```
* 克隆版本库：
```git
git clone https://github.com/udacity/asteroids.git
```
* 查看日志
```git
git log
```
* 要获得彩色输出
```git
git config --global color.ui auto
```
