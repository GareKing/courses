* 创建版本库
* 初始化或创建新的Git代码库
```bash
git init
```
* 查看新建库中的文件
```bash
git log #若果没有文件提交，会报错
```
* 列出当前目录所有还没有被git管理的文件和被git管理且被修改但还未提交(git commit)的文件
```bash
git status #列出库中文件的详细信息
```
* 将文件添加到存储区域
```bash
git add file
```
